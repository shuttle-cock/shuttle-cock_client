# Iconography

<br />
<br />

## 아이콘 네이밍 규칙

- 기능 및 목적 기반: 아이콘의 기능이나 목적을 반영하여 이름을 지정합니다.
- 크기 표시: 크기를 명확히 구분하기 위해 이름에 크기를 포함합니다.
- 케밥 케이스: 단어를 구분하기 위해 하이픈(-)을 사용하여 파일 이름을 명확하게 합니다.

<br />

### 이름 규격 예시

기능/목적:

- cancel
- comment
- ...

크기:

- `-s` for Small
- `-m` for Medium
- `-l` for Large

### 예시 이름

- cancel-s.svg
- cancel-m.svg
- cancel-l.svg
- comment-s.svg
- comment-m.svg
- comment-l.svg
- ...
